- ты работаешь с наитивным js. Ты не используешь Vue Cli, npm, babel, node js, import/export в js и однофайловые компоненты.
- когда начинаешь работу, то в первую очередь проверяй существует ли уже такой файл в git (страница, роутер, app.js, компонент, стили и другие)
	если этот файл существует, то ты интегрируешь новую логику в старую, обновляешь старый файл так, чтобы не нарушить старую логику работы
- если ты видишь, что новая логика не совместима со старой, то переспрашиваешь меня об этом
- названия страниц и их файлов (views) делаем исходя из пути к ним с заглавной буквы, например Home.js, Orders.js, OrdersCreate.js
- домашняя страница - Home.js, страница с error 404 - Error404.js, страница с error 403 - Error403.js
- ты используешь шаблон Velzon, подбирай оттуда элементы интерфейса, которые наиболее соответствуют нашему функционалу.
шаблон находится по адресу - https://themesbrand.com/velzon/html/master/index.html
