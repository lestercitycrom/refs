### Описание структуры:

#### Корневой каталог
- `index.php`: Основная точка входа в приложение, загружает все необходимые ресурсы и инициализирует Vue.js.
- `.htaccess`: Файл конфигурации для перенаправления запросов и работы с маршрутизацией Vue.js.
- `composer.json`: Файл для управления зависимостями PHP с помощью Composer.
- `/vendor`: Папка для зависимостей PHP, управляемых Composer.

#### API
- `/api`: В этом каталоге находятся файлы, связанные с серверной частью API.
  - `api.php`: Главный файл для обработки API-запросов.
  - `/controllers`: Каталог для контроллеров.
    - `ExampleController.php`: Пример контроллера.
  - `/models`: Каталог для моделей.
    - `ExampleModel.php`: Пример модели.
  - `/classes`: Каталог для дополнительных классов.
    - `ExampleClass.php`: Пример дополнительного класса.

#### Статические ресурсы
- `/assets`: Каталог для хранения статических ресурсов (CSS, JS, изображения).
  - `/css`: Каталог для CSS-файлов.
    - `styles.css`: Пример CSS файла.
  - `/js`: Каталог для JavaScript-файлов.
    - `scripts.js`: Пример JavaScript файла.
  - `/img`: Каталог для изображений.
    - `example.png`: Пример изображения.

#### Фронтенд
- `/frontend`: Каталог для хранения фронтенд файлов.
  - `/components`: Каталог для компонентов Vue.js.
    - `ExampleComponent.js`: Пример компонента.
  - `/store`: Каталог для Vuex-хранилища.
    - `store.js`: Основной файл Vuex-хранилища.
  - `/views`: Каталог для представлений Vue.js.
    - `ExampleView.js`: Пример представления.
  - `app.js`: Основной файл инициализации Vue.js приложения.
  - `router.js`: Файл конфигурации маршрутизатора Vue.js.

#### Шаблон
- '/template': каталог для хранения скриптов, стилей, шрифтов и прочих файлов шаблона Velzon, который мы используем для CRM
