### Описание структуры:

#### Корневой каталог
- index.php: Основная точка входа в приложение, загружает все необходимые ресурсы и инициализирует Vue.js.
- .htaccess: Файл конфигурации для перенаправления запросов и работы с маршрутизацией Vue.js.
- composer.json: Файл для управления зависимостями PHP с помощью Composer.
- /vendor: Папка для зависимостей PHP, управляемых Composer.

#### API
/api: В этом каталоге находятся файлы, связанные с серверной частью API.
/api/api.php: Главный файл для обработки API-запросов.
/api/controllers: Каталог для контроллеров.
/api/models: Каталог для моделей.
/api/classes: Каталог для дополнительных классов.


#### Статические ресурсы
/assets: Каталог для хранения статических ресурсов (CSS, JS, изображения).
/assets/css: Каталог для CSS-файлов.
/assets/css/styles.css: Основной файл стилей проекта
/js: Каталог для JavaScript-файлов.
/assets/img: Каталог для изображений.


#### Фронтенд
/frontend: Каталог для хранения фронтенд файлов.
/frontend/components: Каталог для компонентов Vue.js.
/frontend/store: Каталог для Vuex-хранилища.
/frontend/store/store.js: Основной файл Vuex-хранилища.
/frontend/views: Каталог для представлений (страниц) Vue.js.
/frontend/app.js: Основной файл инициализации Vue.js приложения.
/frontend/router.js: Файл конфигурации маршрутизатора Vue.js.

#### Шаблон
- '/template': каталог для хранения скриптов, стилей, шрифтов и прочих файлов шаблона Velzon, который мы используем для CRM
